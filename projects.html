  • Automatic Driveway Light System Using Arduino
    UMass Boston
    Developer and Presenter
    Green Initiative
    • The Arduino Driveway Light System is an automated lighting solution designed to enhance security and energy efficiency in a driveway.
    • Using light and motion sensors, an Arduino microcontroller intelligently controls the illumination of the driveway. During daylight or well-lit conditions, the lights remain off to save energy. In low-light conditions or when motion is detected, the system activates the lights automatically, providing security and visibility.
    • Additionally, the system may include time-based controls, manual override options, and remote monitoring capabilities for user convenience. This project aims to optimize energy usage while ensuring a well-lit and secure environment in the driveway.
